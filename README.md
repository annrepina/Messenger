# ZipMessenger
Это учебный проект, цель которого - улучшить навыки программирования на C#, изучить основы сетевого программирования.

# Краткое ТЗ проекта
Необходимо разработать программу для обмена сообщений с помощью TCP. Точнее комплекс программ: клиент и сервер.

1. Пользователь должен иметь возможность авторизоваться под своим аккаунтом. А также выйти из аккаунта.
2. Пользователь должен видеть список своих бесед с другими пользователями.
3. Пользователь должен иметь возможность искать других пользователей
4. Пользователи должны иметь возможность переключится на конкретную беседу. Беседа - это диалог между двумя пользователями. Групповых бесед делать не следует.
5. При входе в беседу должны отражаться последние сообщения, а также новые сообщения которые пользователь получил пока не был онлайн.
6. Должна быть возможность ввода новых сообщений. При отправке новое сообщение моментально появляется в беседе. Можно добавить статус доставки сообщения.
7. При получении сообщения пользователь должен быть уведомлен.
8. Следует обратить внимание на тот факт, что пользователь может подключится к своему аккаунту с нескольких устройств одновременно.
9. Должна быть возможность удалять свои сообщения.
